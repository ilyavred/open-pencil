# Выделение и манипуляции

Выделяйте объекты, чтобы перемещать, изменять размеры, вращать, дублировать и организовывать их на холсте.

## Выделение

- **Клик** по узлу — выделяет его (снимает выделение со всего остального)
- **Shift + клик** — добавляет или убирает узел из текущего выделения
- **Перетаскивание рамкой** — перетащите по пустому холсту, чтобы нарисовать прямоугольник выделения; все пересекающиеся узлы выделяются при отпускании кнопки
- **⌘ A** — выделить все узлы на текущей странице
- **Клик по пустому холсту** — снять выделение со всех

## Перемещение

- **Перетаскивание** выделенного узла перемещает его (все выделенные узлы двигаются вместе)
- **Клавиши-стрелки** — сдвиг выделенных узлов на 1 px
- **Shift + стрелки** — сдвиг на 10 px

## Изменение размеров

У выделенных узлов отображаются 8 маркеров изменения размера (4 угловых + 4 серединных). Перетащите любой маркер для изменения размера.

- **Shift + перетаскивание** углового маркера сохраняет пропорции

## Вращение

Наведите курсор чуть за пределы углового маркера, чтобы увидеть курсор вращения. Перетаскивайте для поворота.

- **Shift + перетаскивание** фиксирует вращение с шагом 15°

## Дублирование

- **Alt + перетаскивание** (⌥ + перетаскивание на Mac) — дублирует выделенный узел и перемещает копию
- **⌘ D** — дублировать на месте

## Удаление

Нажмите **Backspace** или **Delete**, чтобы удалить все выделенные узлы.

## Z-порядок

Изменение порядка наложения узлов внутри родителя:

- **]** — переместить на передний план (вверх в списке потомков)
- **[** — переместить на задний план (вниз в списке потомков)

## Видимость и блокировка

- **⇧ ⌘ H** — переключить видимость. Скрытые узлы не отображаются, но остаются в панели слоёв.
- **⇧ ⌘ L** — переключить блокировку. Заблокированные узлы нельзя выделить или переместить на холсте.

## Перемещение на страницу

Переместите выделенные узлы на другую страницу через [контекстное меню](./context-menu). Узлы перемещаются под холст целевой страницы.

## Секции

При рисовании секции на холсте перекрывающиеся соседние узлы автоматически становятся дочерними элементами новой секции.

## Сочетания клавиш

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Выделить всё | ⌘ A | Ctrl + A |
| Дублировать | ⌘ D | Ctrl + D |
| Дублировать + переместить | ⌥ + перетаскивание | Alt + перетаскивание |
| Удалить | ⌫ / Delete | Backspace / Delete |
| Сдвиг на 1 px | Клавиши-стрелки | Клавиши-стрелки |
| Сдвиг на 10 px | ⇧ + стрелки | Shift + стрелки |
| На передний план | ] | ] |
| На задний план | [ | [ |
| Переключить видимость | ⇧ ⌘ H | Shift + Ctrl + H |
| Переключить блокировку | ⇧ ⌘ L | Shift + Ctrl + L |

## Советы

- Используйте панель [Слои и страницы](./layers-and-pages), чтобы видеть и менять порядок узлов, когда они перекрываются.
- См. [Контекстное меню](./context-menu) для дополнительных действий, таких как группировка и создание компонентов.
